# Invisi Labs

[English](README.md) · **[繁體中文](README.zh-TW.md)**

> 花幣不留痕，隱私由你掌。

**Invisi Labs** 基於零知識證明，打造隱私優先的支付基礎設施。
旗艦產品 **InvisiPay** 讓任何人都能以隱私方式消費加密貨幣——
覆蓋 180+ 國家、8000 萬家商戶，3 秒內完成結算，自託管，合規可審計。

### 產品

| 產品 | 說明 |
|------|------|
| **U-Card** | 支援 Visa/Mastercard 的加密貨幣簽帳卡，穩定幣消費、3 秒內以當地貨幣結算。 |
| **行動應用** | 非託管錢包，支援掃碼付款、隱私找零管理及外部錢包連接。 |
| **OTC 兌換** | 法幣 ↔ 加密貨幣兌換，支援銀行轉帳、支付寶、微信支付。 |

### 技術

| 支柱 | 說明 |
|------|------|
| **ZK 隱私** | 交易金額與參與方由 Noir ZK 電路遮蔽，鏈上僅存證明與承諾，不暴露明文。 |
| **合規稽核** | View Key 與稽核工具讓監管方與商戶在不洩露隱私的前提下完成驗證。 |
| **基礎設施** | Invisible Protocol — 可複用的跨鏈隱私結算層，支援 Ethereum、Arbitrum、Base 與 Solana。 |
