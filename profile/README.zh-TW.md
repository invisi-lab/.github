> 打破商業隱私邊界，建構無感、合規的結算基礎設施。

**Invisi Labs** 專注於基於零知識證明的隱私結算與支付基礎設施，核心產品 **InvisiPay** 是全球首個基於 **Noir** 驅動的商業隱私結算層，為商戶、跨境企業與 Web3 機構提供隱私優先、鏈上可驗證且合規可審計的結算能力。

### 核心方向

| 方向 | 說明 |
|------|------|
| **隱私優先** | 交易金額與參與方透過 ZK 脫敏，鏈上僅存證明與承諾，不暴露明文。 |
| **無感合規** | 支付體驗與常見收銀一致，同時透過 View Key 等機制滿足監管與審計要求。 |
| **基礎設施** | 可被全行業複用的隱私結算協議與標準，支援多鏈、多資產擴展。 |
