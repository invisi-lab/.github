# Invisi Labs

[English](README.md) · **[简体中文](README.zh-CN.md)** · [繁體中文](README.zh-TW.md)

> 打破商业隐私边界，构建无感、合规的结算基础设施。

**Invisi Labs** 专注于基于零知识证明的隐私结算与支付基础设施，核心产品 **InvisiPay** 是全球首个基于 **Noir** 驱动的商业隐私结算层，为商户、跨境企业与 Web3 机构提供隐私优先、链上可验证且合规可审计的结算能力。

### 核心方向

| 方向 | 说明 |
|------|------|
| **隐私优先** | 交易金额与参与方通过 ZK 脱敏，链上仅存证明与承诺，不暴露明文。 |
| **无感合规** | 支付体验与常见收银一致，同时通过 View Key 等机制满足监管与审计要求。 |
| **基础设施** | 可被全行业复用的隐私结算协议与标准，支持多链、多资产扩展。 |
